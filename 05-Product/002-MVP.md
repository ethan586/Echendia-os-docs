## Product Shape
## 产品形态

The current product is not AI Chat.

当前产品不是 AI Chat。

It is a Founder Operating System.

它是 Founder Operating System。

Chat is only the unified entry point.

聊天只是统一入口。

The product goal remains:

产品目标仍然是：

> Help founders translate long-term goals into daily execution.

> 帮助 Founder 将长期目标转化为每天的执行。

## Personal Layer
## 个性化层

Above Founder Core, there is a Personal Layer.

Founder Core 之上，存在 Personal Layer。

The Personal Layer stores each user's own configuration.

Personal Layer 保存用户自己的配置。

Examples include:

例如：

- Work schedule
- 工作安排

- Meal schedule
- 用餐安排

- Fitness goals
- 健身目标

- Learning goals
- 学习目标

- Financial accounts
- 财务账户

- Daily routines
- 日常流程

These are user data.

这些属于用户数据。

They are not product logic.

不是产品逻辑。

## Product Principles
## 产品原则

- Chat is the interface, not the product.
- Chat 是入口，不是产品。

- Mission drives execution.
- 长期目标驱动每日执行。

- Founder Core should remain universal.
- Founder Core 保持通用。

- Personal Layer should remain configurable.
- Personal Layer 保持可配置。

- Build for daily use, not occasional use.
- 产品应每天都值得打开。
