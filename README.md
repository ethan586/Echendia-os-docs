## Documentation Principles
## 文档原则

- Document decisions, not only ideas.
- 记录决策，而不只是记录想法。

- Keep documents bilingual.
- 保持中英双语。

- Keep structure stable.
- 保持结构稳定。

- Prefer clear and maintainable writing.
- 优先使用清晰、可维护的表达。

- Use GitHub as long-term memory.
- 使用 GitHub 作为长期记忆。

- Sync important discussions through Change Requests.
- 通过 Change Request 同步重要讨论。
