# Toolbox
# 工具箱

## Purpose
## 目的

This directory is the official AI tools and developer tools knowledge base for Echendia.

本目录是 Echendia 官方 AI 工具与开发工具知识库。

It is not a software collection.

它不是软件收藏夹。

It is a long-term maintained tool system for Echendia.

它是 Echendia 长期维护的工具体系。

Only tools that can help Founder, Agent, Engineering, Product, or Business growth should be included.

只有真正能够帮助 Founder、Agent、Engineering、Product 或 Business 增长的工具，才允许进入本目录。

---

## Principles
## 原则

- Tools should be selected for long-term value.
- 工具应基于长期价值选择。

- Tools should support real workflows.
- 工具应支持真实工作流。

- Tools should be reviewed before adoption.
- 工具在采用前应经过评估。

- Tools should not be added only because they are new.
- 不应仅因为工具很新就加入本目录。

- Every tool should follow a consistent documentation template.
- 每个工具都应按照统一模板记录。

---

## Scope
## 范围

This directory covers AI tools, MCP servers, Agent development tools, developer tools, and productivity tools.

本目录覆盖 AI 工具、MCP Server、Agent 开发工具、开发工具以及效率工具。

Future tool updates should be added to the existing documents.

未来发现值得学习的新工具时，应更新现有文档。

Do not create a new Markdown file for every new tool.

不要为每一个新工具单独新建 Markdown 文件。
