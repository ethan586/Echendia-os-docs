# Productivity Tools
# 效率工具

## Purpose
## 目的

This document records productivity tools that may support Founder and daily operating workflows.

本文记录可能支持 Founder 与日常运营工作流的效率工具。

Productivity tools should help reduce friction, not create another system to maintain.

效率工具应减少摩擦，而不是制造另一个需要维护的系统。

## Maintenance Rule
## 维护规则

Add a productivity tool only when it becomes part of a repeatable workflow.

只有当效率工具成为可重复工作流的一部分时，才将它加入本目录。
