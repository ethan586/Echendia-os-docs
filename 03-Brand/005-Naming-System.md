# Naming System
# 命名体系

Version: 1.0

Status: Official

---

## Purpose
## 目的

This document defines the official naming system for Echendia.
本文档定义 Echendia 的官方命名体系。

It helps keep company, brand, product, and system names consistent across all documentation and public communication.
它用于确保公司、品牌、产品和系统命名在所有文档与对外传播中保持一致。

---

## Official Names
## 官方命名

**Echendia** is the company name and brand name.
**Echendia** 是公司名与品牌名。

**Echendia OS** is the first core product of Echendia.
**Echendia OS** 是 Echendia 的第一个核心产品。

**Echendia Runtime** is a possible future AI runtime layer.
**Echendia Runtime** 是未来可能建设的 AI 运行层。

**Echendia Memory** is the long-term memory system.
**Echendia Memory** 是长期记忆系统。

**Echendia Agents** is the future agent system.
**Echendia Agents** 是未来的 Agent 系统。

---

## Usage Principles
## 使用原则

Use **Echendia** when referring to the company, brand, values, mission, philosophy, or long-term belief.
当表达公司、品牌、价值观、使命、哲学或长期信念时，使用 **Echendia**。

Use **Echendia OS** when referring to the specific product, system, feature set, or product capability.
当表达具体产品、系统、功能集合或产品能力时，使用 **Echendia OS**。

Avoid using **Echendia** and **Echendia OS** interchangeably in external communication.
对外传播时，应避免把 **Echendia** 和 **Echendia OS** 混用。

The naming system should remain stable unless the brand architecture changes.
除非品牌架构发生变化，否则该命名体系应保持稳定。
